# ⚙️ Sistema de Recomendação de Imóveis — Back-end

Este projeto foi desenvolvido como parte de um trabalho acadêmico, sendo responsável pelo **back-end** da aplicação de recomendação de imóveis.

O sistema processa os dados fornecidos pelo usuário e utiliza uma lógica de decisão para sugerir as melhores opções de imóveis.

---

## 📌 Objetivo

> Receber dados do usuário, processar as informações e retornar recomendações de imóveis com base em critérios definidos.

O back-end tem como responsabilidade:

* Processamento das informações do usuário
* Aplicação da lógica de recomendação
* Integração com banco de dados
* Comunicação com o front-end

---

## 🧠 O que foi desenvolvido neste projeto

* Estruturação da aplicação em **Java**
* Implementação da lógica de recomendação
* Integração com banco de dados
* Manipulação de dados persistentes
* Comunicação com o front-end

---

## 🛠️ Tecnologias Utilizadas

* **Java** – lógica da aplicação
* **MySQL** – banco de dados relacional

---

## 🖥️ Como rodar o projeto localmente

1. Clone o repositório:

   ```bash
   git clone https://github.com/ArthurVictor42/projeto-5-periodo.git
   ```

2. Configure o banco de dados MySQL

   * Crie o banco conforme o projeto
   * Ajuste as credenciais no código

3. Execute a aplicação Java

   * Via IDE (Eclipse/IntelliJ) ou conforme configuração do projeto
